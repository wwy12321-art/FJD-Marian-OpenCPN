## 编译说明
### 环境要求
- **Android SDK**: API 33
- **Android NDK**: 25.1.8937393
- **CMake**: 3.22.1
- **Java**: OpenJDK 11
- **编译器**: Clang 14.0.6 (Android NDK)

### 编译步骤
执行build-android-libs-native.sh脚本编译
